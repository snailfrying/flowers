# Screenshots

This directory contains screenshots for the Flowers project documentation.

## Screenshots

The following screenshots are available:

1. **翻译.png** - Shows the translation feature when text is selected on a webpage
2. **聊天.png** - Displays the RAG-powered chat interface
3. **总览.png** - Demonstrates the note generation feature from web content (overview)
4. **笔记.png** - Shows the notes management interface

## Screenshot Guidelines

- **Format**: PNG format recommended
- **Size**: Width 800-1200px, maintain aspect ratio
- **Quality**: High resolution, clear and readable
- **Content**: Show actual usage, avoid sensitive information
- **Naming**: Use descriptive names (Chinese or English)

## Adding New Screenshots

1. Take a screenshot of the feature
2. Save it with a descriptive name in this directory
3. Update the README.md to reference the new screenshot
4. Ensure the file size is reasonable (< 500KB recommended)
